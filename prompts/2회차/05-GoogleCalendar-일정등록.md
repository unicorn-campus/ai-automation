# 프롬프트 예제: Google Calendar 회의 일정 자동 등록

## 사용 목적
회의록·메모에서 다음 일정을 추출하여 Google Calendar에 자동 등록.  
일정 누락 방지 및 수동 입력 시간 절감.

---

```text
[목표]
텍스트에서 일정 정보를 추출하여 Google Calendar에 이벤트 자동 등록

[역할]
당신은 임원 일정 관리 경력 8년의 전문 스케줄러입니다.

[맥락]
- 내 상황: 회의록·메모·메일에 언급된 다음 일정을 캘린더에 일일이 입력하는 시간이 낭비됨
- 결과물 독자: 일정 등록 요청자 본인 — 캘린더 확인만으로 모든 약속을 파악하는 용도

[입력정보]
- 일정 추출 대상: 아래 <원문> 태그 안에 붙여 넣기

<원문>
{일정이 포함된 텍스트 (회의록·메모·메일 내용)}
</원문>

- 등록 캘린더: {캘린더명, 기본값: 기본 캘린더}

[작업방법]
1. <원문>에서 일정 후보 탐지
   - 날짜·시간 표현: "다음 주 화요일", "6월 20일 오후 3시" 등
   - 참석자 정보: 이름·팀명·이메일
   - 장소 정보: 회의실명·주소·온라인 링크
   - 안건·제목: 회의 목적 또는 이벤트명
2. 탐지된 일정 목록을 사용자에게 확인 요청 (등록 전 검토)
3. 확인된 일정을 Google Calendar MCP로 등록
   - create_event: 제목·시작/종료시간·참석자·장소·설명 포함
   - suggest_time: 참석자 일정 충돌 시 대안 시간 제안
4. 등록 완료 목록 출력
- 톤앤매너: 간결한 업무 확인 스타일, 명사체 종결
- 작성 규칙:
  - 날짜·시간 모호 시 "확인 필요"로 표기 후 등록 보류
  - 종료 시간 미언급 시 기본 1시간으로 설정

[출력]
- 응답으로 출력 (마크다운 형식)
- 구성 순서: 탐지된 일정 목록(확인 요청) → 등록 완료 목록

[제약조건]
- MUST: 등록 전 탐지 목록 출력 후 확인 단계 거칠 것
- MUST NOT: 날짜·시간 불명확한 일정 임의 추정 등록 금지
- 완료조건: 등록 완료 이벤트 목록과 Calendar 링크가 출력됨

[예시]
(입력)
다음 주 목요일 오후 2시에 설계 검토 회의 하겠습니다. 참석자는 강동철, 이준호이고
회의실 A에서 진행합니다. 그리고 금요일 오전에 협력사 미팅도 잡아야 해요.

(출력)
## 탐지된 일정 (등록 전 확인)

| # | 제목 | 날짜·시간 | 참석자 | 장소 | 확인 필요 |
|---|------|----------|--------|------|----------|
| 1 | 설계 검토 회의 | 2026-06-18 (목) 14:00~15:00 | 강동철, 이준호 | 회의실 A | - |
| 2 | 협력사 미팅 | 2026-06-19 (금) 오전 | 미상 | 미상 | 시간·참석자 확인 필요 |

등록하시겠습니까? (1번만 / 전체 / 취소)
```

---

## 활용 팁
- 회의록(02-표준회의록) 결과물을 `<원문>`에 붙여 넣으면 액션 일정 자동 등록 가능
- `suggest_time` 도구로 참석자 가능 시간대 자동 탐색 가능
- ArtBridge 공연 추천 결과의 공연 일정도 동일 방식으로 캘린더 등록 가능
