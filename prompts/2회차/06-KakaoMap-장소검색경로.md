# 프롬프트 예제: KakaoMap 장소 검색 및 경로 안내

## 사용 목적
목적지 주변 장소를 검색하고 출발지에서 대중교통·도보 경로를 함께 제공.  
공연 관람·협력사 방문·팀 회식 장소 탐색에 즉시 활용 가능.

---

```text
[목표]
키워드로 목적지 주변 장소를 검색하고 출발지에서의 이동 경로 안내 보고서 작성

[역할]
당신은 도시 이동 경험 최적화 전문가입니다.

[맥락]
- 내 상황: 낯선 장소 방문 전 주변 식당·카페·주차장을 미리 파악하고 이동 경로도 확인 필요
- 결과물 독자: 방문 예정자 본인 또는 팀원 — 이동·식사 계획을 한 번에 확인하는 용도

[입력정보]
- 이동 조건: 아래 <이동조건> 태그 안에 입력

<이동조건>
출발지: {주소 또는 장소명}
목적지: {주소 또는 장소명}
이동 수단: {대중교통/도보/자전거}
주변 검색 키워드: {맛집/카페/주차장/편의점 등}
방문 인원: {N명}
</이동조건>

[작업방법]
1. KakaoMap MCP로 장소 검색
   - SearchPlaceByKeywordOpen: 목적지 주변 키워드 검색 (반경 500m 기준)
2. 이동 경로 조회
   - GetPublicTransitDirections: 대중교통 선택 시 — 소요시간·환승 횟수·요금 포함
   - GetWalkDirections: 도보 선택 시 — 소요시간·거리 포함
   - GetBikeDirections: 자전거 선택 시 — 소요시간·거리 포함
3. 주변 검색 결과를 다음 기준으로 정렬
   - 목적지와의 거리 (가까운 순)
   - 방문 인원 수용 가능 여부 (N명 이상)
4. 상위 3개 장소 상세 정보 정리
- 톤앤매너: 실용적·친근한 여행 안내 스타일
- 작성 규칙:
  - 장소명·주소·거리·영업시간 필수 포함
  - 이동 경로는 단계별(탑승역→환승→하차) 형식으로 표기

[출력]
- 응답으로 출력 (마크다운 형식)
- 구성 순서: 이동 경로 요약 → 주변 추천 장소 TOP 3

[제약조건]
- MUST: 이동 소요시간 반드시 포함
- MUST NOT: KakaoMap 데이터에 없는 장소 임의 추가 금지
- 완료조건: 이동 경로 요약과 주변 장소 TOP 3이 모두 출력됨

[예시]
(입력)
출발지: 강남역 / 목적지: 블루스퀘어 (한남동) / 이동 수단: 대중교통
주변 검색: 맛집 / 방문 인원: 6명

(출력)
## 이동 경로

**강남역 → 블루스퀘어** (대중교통, 약 25분)
1. 강남역 6호선 승차 → 이태원역 하차 (3정거장, 10분)
2. 이태원역 1번 출구 도보 8분
- 요금: 1,400원

## 주변 맛집 TOP 3 (블루스퀘어 반경 500m)

| 순위 | 장소명 | 거리 | 영업시간 |
|------|--------|------|----------|
| 1 | 이태원 갈비집 | 150m | 11:00~22:00 |
| 2 | 한남 브런치카페 | 230m | 09:00~21:00 |
| 3 | 이탈리안 레스토랑 | 350m | 12:00~22:00 |
```

---

## 활용 팁
- ArtBridge 공연 추천(04) 결과의 공연장을 목적지로 그대로 연결 가능
- 협력사 방문 시 주변 주차장 검색 키워드로 "주차장" 입력
- 팀 회식 장소 탐색 시 방문 인원을 팀원 수로 설정하면 수용 가능 식당만 필터링
