# 프롬프트 예제: 찐맛집 MCP 지역 맛집 Top 5 비교

## 사용 목적
광고·협찬 리뷰를 제거한 신뢰도 높은 데이터로 특정 지역의 맛집 Top 5를 추천받고,  
1·2위를 심층 비교하여 최종 선택을 돕는 프롬프트.

---

```text
[목표]
찐맛집 MCP로 지역·음식 키워드 기반 맛집 Top 5를 추천받고, 1·2위 비교 분석 보고서 작성

[역할]
당신은 광고 없는 찐 리뷰 데이터를 분석하는 맛집 큐레이터입니다.

[맥락]
- 내 상황: 팀 회식·방문 지역 식사 등 모임 장소를 선택해야 하는 상황
- 결과물 독자: 장소 결정자 본인 또는 팀원 — 신뢰할 수 있는 데이터 기반으로 신속하게 선택하는 용도

[입력정보]
- 검색 조건: 아래 <검색조건> 태그 안에 입력

<검색조건>
지역: {시·구·동 단위 입력 예: 강남구, 판교, 서울 마포구}
음식 키워드: {삼겹살 / 초밥 / 파스타 / 한식 등}
우선 고려 사항: {맛 / 웨이팅 / 분위기 / 가격 중 중점 항목}
</검색조건>

[작업방법]
1. 찐맛집 MCP `jjmz-recommend_region` 호출
   - region: 지역 입력값
   - food_keyword: 음식 키워드 입력값
   - focus: 우선 고려 사항 입력값
   - topn: 5 (고정)
2. 반환된 Top 5 결과를 점수·시그널 기준으로 표로 정리
3. 찐맛집 MCP `jjmz-compare_places` 호출 — 1위 vs 2위 비교
   - query_a: 1위 가게명 + 지역
   - query_b: 2위 가게명 + 지역
   - focus: 우선 고려 사항 입력값
4. 비교 결과를 축별(맛·분위기·웨이팅·위생·서비스·가격) 표로 정리
5. 종합 추천 1곳 선정 및 선정 이유 2줄 이내 작성
- 톤앤매너: 간결하고 실용적인 선택 도우미 스타일
- 작성 규칙:
  - 광고 제거율(Exclusion rate) 반드시 표기
  - 점수는 10점 만점 기준으로 표기
  - 주소는 도로명 주소로 표기

[출력]
- 응답으로 출력 (마크다운 형식)
- 구성 순서:
  1. Top 5 추천 표
  2. 1위 vs 2위 축별 비교 표
  3. 최종 추천 및 이유

[제약조건]
- MUST: 광고·협찬 제거 후 산정된 점수임을 상단에 명시
- MUST NOT: MCP 반환 데이터에 없는 가게 임의 추가 금지
- 완료조건: Top 5 표 + 1·2위 비교 표 + 최종 추천 모두 출력됨

[예시]
(입력)
지역: 강남구 / 음식 키워드: 삼겹살 / 우선 고려 사항: 맛

(출력)
> 광고·협찬 리뷰 제거 후 점수 산정 | 평가 기준: 맛 23% / 분위기 20% / 웨이팅 18% / 위생 17% / 서비스 17% / 가격 5%

## 강남구 삼겹살 맛집 Top 5

| 순위 | 가게명 | 점수 | 맛 신호 | 광고제거율 | 주소 |
|------|--------|------|---------|-----------|------|
| 1 | 봉우화로 | 5.9 | +7 (pos 15/neg 8) | 3.3% | 봉은사로18길 76 |
| 2 | 고기꾼김춘배 강남점 | 5.8 | +9 (pos 10/neg 1) | 0.0% | 강남대로100길 13 |
| 3 | 다몽집 신논현본점 | 5.7 | +13 (pos 15/neg 2) | 0.0% | 강남대로100길 13 지하 |
| 4 | (4위 가게명) | X.X | ... | ...% | ... |
| 5 | (5위 가게명) | X.X | ... | ...% | ... |

## 1위 vs 2위 비교 (맛·웨이팅 집중)

| 평가 항목 | 봉우화로 | 고기꾼김춘배 |
|----------|----------|-------------|
| 맛 | pos 15 / neg 8 | pos 10 / neg 1 ✓ |
| 웨이팅 | 데이터 없음 | 데이터 없음 |
| 분위기 | pos 7 / neg 0 ✓ | pos 5 / neg 0 |
| 위생 | pos 5 / neg 4 | 데이터 없음 |
| 종합 점수 | **5.9 ✓** | 5.8 |

## 최종 추천: 봉우화로

분위기 긍정 신호(+7)가 압도적이며 종합 점수 1위.  
위생 부정 언급(-4)이 있으므로 청결 민감 시 2위 고기꾼김춘배 고려.
```

---

## 활용 팁
- 우선 고려 사항을 "웨이팅"으로 설정하면 대기 시간 민감한 상황에 최적화된 결과 도출 가능
- KakaoMap 장소 검색(06) 프롬프트와 연결하면 추천 맛집까지의 이동 경로를 바로 확인 가능
- 팀원 투표가 필요하면 Top 5 결과를 Slack 공유(07) 프롬프트로 바로 전달 가능
