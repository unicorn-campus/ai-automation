# 프롬프트 예제: Gmail 액션 메일 카카오톡 알림

## 사용 목적
Gmail 수신 메일에서 액션아이템을 추출하여 카카오톡 나챗방으로 요약 알림 전송.  
200자 제한에 맞춰 긴급 메일 제목만 표시하고 나머지는 건수 + Gmail 링크 제공.

---

```text
[목표]
Gmail 수신 메일을 분석하여 액션아이템 요약을 카카오톡 나챗방에 200자 이내로 전송

[역할]
당신은 임원 비서 경력 10년의 전문 업무 조율자입니다.

[맥락]
- 내 상황: 외근·이동 중에도 처리 필요 메일을 카카오톡으로 즉시 파악해야 함
- 결과물 독자: 메일 수신자 본인 — 카카오톡 한 화면에서 오늘 액션 메일을 빠르게 파악

[입력정보]
- 검색 조건: 아래 <검색조건> 태그 안에 입력

<검색조건>
기간: 최근 {N}일
읽지 않은 메일: {예/아니오}
발신자 필터: {특정 도메인 또는 이름, 없으면 "전체"}
</검색조건>

[작업방법]
1. Gmail MCP로 메일 수집
   - search_threads: 검색 조건에 맞는 스레드 조회
   - get_thread: 각 스레드 상세 내용 읽기
2. 각 메일에서 액션아이템 탐지 후 우선순위 분류
   - 긴급: 기한이 오늘~내일 또는 즉시 처리 요청
   - 일반: 기한이 이번 주 이내
   - 검토: 기한 미명시 또는 참고성
   - 불필요: 뉴스레터·자동 발송
3. 카카오톡 메시지 작성 (200자 이내 엄수)
   - 메시지 형식:
     ```
     📬 메일 액션 ({MM/DD HH:MM})
     🔴긴급 {N}건
     {• 제목 (20자 초과 시 말줄임)}
     🟡일반 {N}건 | 🟢검토 {N}건
     ✉️ https://mail.google.com
     ```
   - 긴급 메일: 제목을 건별로 표시 (20자 초과 시 "…"으로 말줄임)
   - 일반·검토: 건수만 표시 (제목 생략)
   - 긴급 메일이 0건이면 "🔴긴급 없음" 표시
   - 긴급 메일이 많아 200자 초과 시: 상위 3건만 표시 후 "외 {N}건" 추가
4. KakaotalkChat-MemoChat MCP로 메시지 전송

[출력]
- 카카오톡 나챗방에 메시지 전송 완료
- 응답으로 전송한 메시지 내용과 전송 결과 출력

[제약조건]
- MUST: 카카오톡 메시지는 200자 이내 (공백 포함)
- MUST: 긴급 메일 제목 표시 시 발신자 개인정보(이름·이메일) 포함 금지
- MUST NOT: 메일 본문 내용을 카카오톡 메시지에 포함 금지
- 완료조건: KakaotalkChat-MemoChat 전송 성공 응답 수신 + 전송 메시지 내용 출력

[예시]
(카카오톡 전송 메시지 예시)
📬 메일 액션 (06/17 08:30)
🔴긴급 2건
• [결재] 설계변경안 승인 요청
• 협력사 납기 일정 확인 요청
🟡일반 3건 | 🟢검토 5건
✉️ https://mail.google.com
```

---

## 활용 팁
- 매일 08:30 원격 루틴으로 설정하면 출근 전 카톡으로 오늘 액션 메일 파악 가능
- `<검색조건>` 기간을 `1일`·`읽지 않은 메일: 예`로 고정하면 완전 자동화
- Gmail 링크 클릭 시 수신함으로 바로 이동하여 상세 내용 확인 가능
- `02-Gmail-메일액션추출.md`와 함께 사용하면 상세 목록(응답) + 카톡 요약(알림) 이중 활용 가능
