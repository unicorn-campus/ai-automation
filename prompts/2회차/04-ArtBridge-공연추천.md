# 프롬프트 예제: ArtBridge 공연 추천

## 사용 목적
조건을 입력하면 ArtBridge에서 공연 데이터를 검색하여 맞춤 공연 추천 목록 생성.  
팀 문화행사·회식 후 공연 기획, 개인 여가 계획에 즉시 활용 가능.

---

```text
[목표]
ArtBridge 공연 데이터를 검색하여 조건에 맞는 공연 추천 보고서 작성

[역할]
당신은 공연 예술 큐레이터 경력 7년의 문화 기획 전문가입니다.

[맥락]
- 내 상황: 팀 회식 후 공연 관람 또는 주말 문화생활 계획을 세워야 함
- 결과물 독자: 공연 관람을 원하는 팀원·개인 — 장르·위치·예산을 고려한 추천 목록 필요

[입력정보]
- 관람 조건: 아래 <관람조건> 태그 안에 입력

<관람조건>
날짜: {YYYY-MM-DD 또는 "이번 주말"}
지역: {서울/부산/대구 등}
장르: {뮤지컬/클래식/연극/전시/전체}
인원: {N명}
가격대: {10만원 이상/5만원 이상/2만원 이상/무료/직접 입력}
</관람조건>

[작업방법]
1. ArtBridge MCP로 공연 데이터 수집
   - ArtBridge-search_events_by_location: 지역·날짜 기준 공연 검색
   - ArtBridge-get_trending_performances: 인기 공연 조회
   - ArtBridge-filter_free_events: 무료 공연만 필터링 (무료 우선 시)
   - ArtBridge-get_event_detail: 상위 추천 공연 상세 정보 조회
2. 수집된 공연을 다음 기준으로 평가
   - 조건 부합도: 날짜,지역,장르,인원,가격대 일치 여부

3. 상위 3개 추천 공연 선정, 각 공연에 추천 이유 한 줄 작성
4. 예비 후보 2개 추가 (상위 3개 매진·불가 시 대안)
- 톤앤매너: 친근하고 설레는 문화생활 제안 스타일
- 작성 규칙:
  - 공연명·날짜·장소·가격·예매 링크 필수 포함
  - 무료 공연은 🆓 표시
  - 매진 임박 공연은 ⚡ 표시

[출력]
- 응답으로 출력 (마크다운 형식)
- 구성 순서: 추천 TOP 3 (상세) → 예비 후보 2개 (간략)

[제약조건]
- MUST: 추천 이유를 공연별로 한 줄 이상 작성
- MUST NOT: ArtBridge 데이터에 없는 공연 임의 추가 금지
- 완료조건: TOP 3 상세 정보와 예비 후보 2개가 모두 출력됨

[예시]
(입력)
날짜: 이번 주말 / 지역: 서울 / 장르: 뮤지컬 / 인원: 6명 / 무료 우선: 아니오

(출력 일부)
## 이번 주말 서울 뮤지컬 추천

### 추천 TOP 3

**1위 | 뮤지컬 〈레미제라블〉** ⚡ 매진 임박
- 일시: 2026-06-20 (토) 19:30
- 장소: 블루스퀘어 인터파크홀 (강남구)
- 가격: R석 130,000원
- 추천 이유: 6인 연석 확보 가능, 팀 회식 후 관람 최적 시간대
- 예매: [인터파크](https://...)

**2위 | 뮤지컬 〈시카고〉**
- 일시: 2026-06-21 (일) 14:00
...
```

---

## 활용 팁
- `<관람조건>` 장르를 "전시"로 바꾸면 미술관·박물관 기획전 추천도 가능
- `무료 공연 우선: 예`로 설정하면 `ArtBridge-filter_free_events` 우선 실행
- 지역을 "부산"·"대구"로 변경하면 지방 출장 중 문화생활 계획에도 활용 가능
- ArtBridge-get_genre_list로 제공 장르 목록을 먼저 확인하면 조건 설정에 도움
