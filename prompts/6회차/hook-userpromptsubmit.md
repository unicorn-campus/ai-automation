[목표]
사용자 프롬프트 내 신용카드 번호·주민등록번호를 감지하면 제출을 차단하는
`UserPromptSubmit` 훅을 추가함.
[역할]
당신은 하네스 엔지니어입니다.
[맥락]
- 내 상황: 보안을 위해 플러그인 사용자가 개인정보를 프롬프트로 전송하는것을 차단해야 함
- 결과물 독자: 플러그인 사용자
[입력정보]
- Claude Hook 가이드: references/hook-guide.md
[작업방법]
- Claude Hook 가이드 이해 
- 훅 스크립트 `hooks/block-pii.mjs` 작성:
   - 프롬프트에서 신용카드와 주민등록 번호 감지
   - 감지 시 차단 이유 출력. 이때 개인정보는 마스킹 처리해서 표시 
- 훅 등록:`plugin.json`에 등록
- 패치버전 올리고 플러그인 업데이트
- 개발 완료 보고. Claude Code 재시작과 테스트 방법 안내
[출력]
- `hooks/block-pii.mjs`
[제약조건]
- MUST: 추가정보나 의사결정이 필요하면 사용자에게 요청
- MUST NOT: 
  - 정상 작업 흐름을 막지 않음
  - Hook 테스트는 하지 말고 사용자에게 요청 
- 완료조건: 출력파일 모두 생성
