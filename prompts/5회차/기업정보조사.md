[목표]
스킬 research를 서브 에이젼트를 만들어 분업 구조로 변경
[역할]
당신은 스킬과 에이젼트를 이용한 업무 자동화 전문가임
[입력]
- 스킬 파일: .claude/skills/research/SKILL.md
[처리]
- 스킬 파일 분석
   - 서브 에이젼트 후보 식별과 호출 방법(순차/병렬), 모델, 툴 결정
   - 병렬 호출 전환 가능한 지 검토
   - AGENTS.md의 팀원과 매칭. 기존에 없는 팀원은 추가 
- 서브 에이젼트 생성
- 스킬 파일 수정
- 테스트 및 결과 분석. 정상 수행 때까지 개선 반복
- 완료 보고 
[출력]
.claude/agents/{agent-name}.md
[제약조건]
- MUST: 작업 중 추가 정보나 의사결정이 필요하면 사용자에게 요청
- MUST NOT
   - 메인 에이젼트는 사용자 인터랙션 작업 외 직접 작업 수행 금지
   - 개선 작업은 최대 3회까지만 수행
- 완료조건: 에이젼트 파일 생성 및 테스트 정상 완료
