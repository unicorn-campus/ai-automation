1. 과제 수행 방법 안내
1) 사전설치 
- 가이드: https://github.com/unicorn-plugins/npd/blob/main/resources/guides/setup/prepare.md
- AI툴 중 Claude Desktop 만 설치

2) 과제: 교재 18페이지 
- ① 핵심 3줄 요약
  - 사용자가 제공한 텍스트 또는 소스 파일 경로를 읽어 핵심 요약
   - 소스 파일의 타입은 제한 없음 
- ② 표준 회의록
  - 사용자가 제공한 회의 내용 메모 텍스트 또는 소스 파일 경로을 읽어 표준 회의록으로 정리
   - 소스 파일의 타입은 제한 없음
   - 표준 회의록의 형식은 자유롭게 정의: 결정 사항, Action plan은 필수 포함 
- ③ 시장조사
  - 특정 분야의 시장조사 수행 
※ 교육에서 가이드한 8대 섹션 활용한 프롬프트 작성 

3) 작업 방법 안내 
각 과제별로 아래 단계별로 수행하세요.  
- 작업 디렉토리 작성
  - 사용자 홈 밑에 workspace 생성(최초 1회만 수행): mkdir workspace 
   - 과제별 작업 디렉토리(=프로젝트 디렉토리) 작성
     - cd workspace 
      - mkdir {디렉토리명}
-  작업 디렉토리 이동: cd ~/workspace/{디렉토리명} 
- vscode에서 열기: code . 
- 공통 지침 작성
  - vscode 루트 밑에 파일 2개 생성: AGENTS.md, CLAUDE.md
  - 아래 AGENTS.md 예시의 내용 복사하여 AGENTS.md에 붙여넣기 
 https://github.com/unicorn-campus/law-consult/blob/main/AGENTS.md
  - 팀원 중 오케스트레이터 빼고 모두 지움
   - CLAUDE.md 파일 내용을 ‘@AGENTS.md’로 작성
   - AGENTS.md, CLAUDE.md 저장 
   - Claude Code에 팀원 구성 요청 
     - 폴더를 위에서 생성한 디렉토리로 지정
      - 아래 프롬프트 입력
     “목표에 맞는 팀원을 구성해요. 오케스트레이트 참조. 오케스트레이터의 역할과 책임도 목표에 맞게 수정해요. ”
      - AGENTS.md에 팀원 구성 확인 및 저장 
   - 새 대화창 시작하고 팀원들 인사시키기: 안해도 되지만 팀원들이 정상 등록되었는지 확인용
    "각자 간략하게 자기 소개와 각오 한마디씩 합시다."
- 과제 수행 프롬프트 작성    
  - vscode에서 prompts 디렉토리 만들고 그 아래에 프롬프트 파일 생성(확장자는 txt)
  - 프롬프트 파일에 8대 섹션별로 내용 작성 후 저장 
  - 내용 복사하여 클로드 코드 대화창에 붙여 넣어서 수행 
  ※ 결과를 파일로 바로 작성 시킬 수 있음. '[출력]'섹션에서 결과 파일 경로를 지정
