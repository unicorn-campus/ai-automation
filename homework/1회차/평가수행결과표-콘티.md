## 채점 근거 메모

- **공통과제 미제출 페널티**: 3종 미제출자(신광조 공통 3종, 전두현 회의록·시장조사)는 해당 과제를 0점으로 ①②③ 평균에 반영함.  
  · 신광조: 공통 3종 전부 미제출 → ①②③=0, ④(협약서 검토)만 채점.  
  · 전두현: 요약만 제출 → ①②③ = (요약점수 + 0 + 0) / 3 반영.
- **결과품질(③) 예측 신호**: 실제 실행 없이 산출형식 규정 구체성·지시 명확성·예외/환각 처리·입력→출력 매핑 누락위험으로 예측 채점함.  
  기준 시나리오(요약/회의록/시장조사)를 넣었을 때 예상 산출물의 유용성·정확성·충실성을 기준으로 함.
- **추가과제(④)**: 본인 업무 프롬프트의 실무 적합성·구체성으로 채점(제출만으로 만점 아님). 미제출=0.
- **상위권 특징**: 위임경로·게이트·XML 입력분리·완료조건 체크리스트를 갖춘 김만수(93.4)·최종현(93.0)·신원석(89.2)이 4종 균형으로 최상위.
- **아쉬운 케이스**: 오윤미·이채림·홍지훈·이재성 등은 공통과제 품질이 상위권이나 추가과제 미제출(④=0)로 종합 손실이 큼.
