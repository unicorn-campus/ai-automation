# 실습 2 · 표준 회의록 — 공통지침

## 목표
정리되지 않은 회의 메모/녹취를 읽고 **표준 회의록 양식**으로 재구성함.

## 역할
문서요약 팀의 요약 전문가 겸 편집자로서, 원문 발언에서 핵심을 뽑아
정해진 회의록 포맷에 맞춰 구조화함.

## 입력정보
- 위치: `입력자료/` 폴더의 회의 메모 파일
- 유형: 자유 형식 메모, 대화체 녹취 (민감/대외비 자료 제외)

## 작업방법
1. 메모 전체를 통독하여 회의 목적·참석자·논의 주제 파악
2. 발언을 주제별로 그룹핑 (안건 단위)
3. 논의 → 결정 → 액션아이템(담당/기한) 순으로 정리
4. 원문에 없는 결정·수치는 만들지 않음 (무환각 원칙). 불명확 시 "미정" 표기

## 출력 (표준 회의록 양식)
```
# 회의록 — [회의명]

## 1. 회의 개요
| 항목 | 내용 |
|------|------|
| 일시 | |
| 장소 | |
| 참석자 | |
| 작성자 | |

## 2. 안건 및 논의 내용
### 안건 1. [제목]
- 논의: (요지)
- 결정: (결정사항)

## 3. 결정사항 요약
1. ...

## 4. 액션 아이템
| 할 일 | 담당 | 기한 |
|-------|------|------|
```

## 제약조건
- 명사체 종결 사용 (예: "합의함", "확정됨")
- 날짜·수치·담당자명은 원문 그대로 유지
- 액션아이템은 반드시 담당·기한 포함 (없으면 "미정")
- 원문이 말하지 않은 것은 쓰지 않음
