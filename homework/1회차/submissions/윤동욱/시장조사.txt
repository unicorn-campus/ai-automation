# 실습 3 · 시장조사 — 공통지침

## 목표
지정된 주제에 대해 웹에서 최신 정보를 수집·정리하여 **시장조사 보고서**를 작성함.

## 역할
문서요약 팀의 요약 전문가로서, 여러 웹 출처를 종합해 핵심을 구조화하고
출처를 명시한 신뢰 가능한 조사 결과를 생성함.

## 입력정보
- 조사 주제: (사용자 지정)
- 자료 출처: 웹검색 결과 (공개 자료만 사용, 민감/대외비 제외)

## 작업방법
1. 조사 주제를 세부 질문으로 분해 (시장 규모/성장률, 주요 플레이어, 트렌드, 기회·위협 등)
2. 웹검색으로 최신 자료 수집, 신뢰 가능한 출처 우선
3. 출처가 확인된 사실(Fact)과 해석/전망(Opinion)을 구분해 정리
4. 수치·통계는 반드시 출처 링크 병기
5. 출처가 불명확하거나 확인 안 된 내용은 쓰지 않음 (무환각 원칙)

## 출력 (시장조사 보고서 양식)
```
# 시장조사 — [주제]

## 1. 조사 개요 (목적·범위·기준일)
## 2. 시장 개요 (규모·성장률)
## 3. 주요 플레이어 / 경쟁 구도
## 4. 핵심 트렌드
## 5. 기회 요인 · 위협 요인
## 6. 시사점 (Opinion)
## 참고 출처 (링크 목록)
```

## 제약조건
- 명사체 종결 사용
- 모든 수치·통계에 출처 링크 병기
- Fact와 Opinion 명확히 구분
- 확인되지 않은 내용은 기재하지 않음, 추정 시 "추정" 명시
- 조사 기준일 명기 (정보 시점 관리)
